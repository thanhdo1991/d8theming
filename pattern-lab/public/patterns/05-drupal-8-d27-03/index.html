How to update drupal 8

Step 1: Go to page: https://www.drupal.org/8/download and download lastest version

Step 2: Go to drupal installation and remove core, vendor and all of files in root folder

Step 3: Copy code from new version drupal (downloaded from step 1) to root folder (only copy core, vendor folder and all files in root folder)

Step 4: Run update.php or drush up command
